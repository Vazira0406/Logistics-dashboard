# Logistics-dashboard
This project provides insights into carrier performance, product categories, delivery reliability, cost efficiency, delays, and seasonal patterns — useful for logistics optimization, supply chain managers, and freight forwarders.




## Key Features & Visualizations

- **Total shipment value & top product** — Furniture leads with ~$97k+ cumulative cost
- **Carrier performance** — UPS, FedEx, DHL, LogistiCo, TransAsia
  - Delivery success rate
  - Delayed shipment count
  - Cost efficiency score
- **Delay analysis** — ~20–30% delayed depending on carrier
- **Product breakdown** — Furniture, Electronics, Pharmaceuticals, Clothes, Food
- **Monthly trends** — Shipment volume and delays by month (Jan–Oct 2024)
- **Urgent Attention Index** & **Cost Efficiency Score** — highlighting carriers/products needing optimization

## Tech Stack 


- **Visualization** — Power BI 
- **Dashboard** — Power BI Desktop
